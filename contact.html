Plugin support forum

The official WordPress.org support forum for Simple JWT Login. Great for plugin-specific questions that the broader WordPress community can benefit from.

Visit the forum →
Email support was discontinued on 1 December 2024. Please use Discord, GitHub, or the WordPress forum for all support requests — responses there are faster and help the whole community.

What to expect

Quick questions

Discord is your best bet — conversations happen fast and the community is active.

Bugs & reproducible issues

Open a GitHub issue with steps to reproduce. Include your PHP version, WordPress version, and plugin version.

Feature requests

Submit a GitHub issue tagged "enhancement". Upvotes and comments help prioritise what gets built next.

General help

The WordPress forum is great for configuration questions that future users might search for too.
