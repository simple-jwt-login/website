The version `3.4.4` has been released today on [WordPress](https://wordpress.org/plugins/simple-jwt-login).

This update brings new hooks for customizing responses, a fix for empty JWT headers, and a new OpenAPI spec.
<!--truncate-->

## New Features

- New hooks have been added to allow customization of all success responses
- New [OpenAPI](https://github.com/nicumicle/simple-jwt-login/blob/master/postman/openapi.yaml) spec file has been added

## Bug Fixes

- Do not add empty JWT to the Authorization header

## Compatibility

- The plugin has been tested with WordPress 5.9

If you encounter any issues after updating, please open an [issue](https://github.com/nicumicle/simple-jwt-login/issues) on GitHub.
