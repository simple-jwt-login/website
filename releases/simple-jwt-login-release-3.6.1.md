The version `3.6.1` has been released today on [WordPress](https://wordpress.org/plugins/simple-jwt-login).

This update strengthens the Protect Endpoints feature with JWT validation and restores backward compatibility for existing configurations.

## New Features[​](#new-features "Direct link to New Features")

* Validate JWT on protected endpoints [#141](https://github.com/nicumicle/simple-jwt-login/issues/141)

## Bug Fixes[​](#bug-fixes "Direct link to Bug Fixes")

* Fix backward compatibility for Protect Endpoints - adds `match` / `start_with` matching options for endpoint rules [#143](https://github.com/nicumicle/simple-jwt-login/issues/143)

Read more about Protect Endpoints [here](/docs/protect-endpoints.md).

***

If you encounter any issues after updating, please open an [issue](https://github.com/nicumicle/simple-jwt-login/issues) on GitHub.
