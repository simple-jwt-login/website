This add-on allows you to export your Simple-JWT-Login plugin settings and import them into another website.

</p>


## Installation

1. Download the add-on from [https://github.com/simple-jwt-login/export-import/](https://github.com/simple-jwt-login/export-import/)
2. Upload the zip file into your WordPress
3. Activate the plugin
4. Export or import data

## Export and Import Data

1. Click on "Export"
2. Copy the generated code
3. Login to the new website where you want to import the data
4. Paste the generated code
5. Click on import
6. A confirmation will be prompted. Click Ok.
7. Done
