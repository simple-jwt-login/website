How it works

Four steps to a perfect copy

1

Install on both sites

Install Simple JWT Login and the Export-Import add-on on both the source and destination WordPress sites.

2

Export from source

Go to Simple JWT Login settings on the source site and click Export to download your configuration.

3

Review the diff

On the destination site paste the config. The add-on shows a diff of every change before you confirm.

4

Confirm & done

Click Import. All settings, Auth Codes, and rules are applied instantly — no page reloads needed.

Use Cases

Any time consistency matters

Staging → production

Perfect your configuration on staging, export it in one click, and apply it to production. No risk of config drift, no manual re-entry of settings.

Multi-site replication

Running a WordPress network? Export once and import to every node. Keep all your sites in sync with a single source of truth.

Backup & restore

Before a major update or plugin change, export your configuration as a safety net. Restore it in seconds if anything goes wrong.

Getting Started

Up and running in 4 steps

1
Install Simple JWT Login

Make sure the core plugin is installed and active on all WordPress sites involved.

2
Download the add-on

Grab the zip from GitHub and upload it via Plugins → Add New → Upload Plugin on each site.

3
Export your configuration

On the source site go to Simple JWT Login settings and click Export. Save the generated JSON.

4
Import on the destination

On the destination site paste the JSON into the Import field, review the diff, and confirm.
